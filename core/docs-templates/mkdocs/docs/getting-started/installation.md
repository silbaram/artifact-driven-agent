# 설치

## 사전 요구사항

필요한 의존성 및 환경 설정 목록을 확인하세요.

## 설치 방법

### npm 사용

```bash
npm install [package-name]
```

### yarn 사용

```bash
yarn add [package-name]
```

### 소스 코드에서 설치

```bash
git clone https://github.com/username/repo.git
cd repo
npm install
npm run build
```

## 설치 확인

설치가 제대로 되었는지 확인하세요:

```bash
[command] --version
```
