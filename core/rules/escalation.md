# Escalation Rules (에스컬레이션 규칙)

> 이 문서는 Manager에게 보고해야 하는 상황과 절차를 정의한다.
> 모든 역할이 이 규칙을 따른다.

---

## 1. 에스컬레이션 원칙

- 판단 불가 시 → 즉시 에스컬레이션
- 규칙 외 상황 → 즉시 에스컬레이션
- 작업 완료 후 이상 발견 → 보고
- 혼자 결정하지 않음

---

## 2. 즉시 에스컬레이션 (BLOCK)

> 작업을 **중단**하고 Manager에게 보고해야 하는 상황

### 2.1 BLOCK 상황

| 상황 | 예시 |
|------|------|
| project.md에 없는 기술 필요 | 새 라이브러리 도입 필요 |
| 요구사항 모호/충돌 | plan.md 해석 불가 |
| 문서 간 충돌 | plan.md ↔ project.md 불일치 |
| 역할 간 해석 불일치 | 같은 문서 다르게 해석 |
| 구현 불가능 판단 | 기술적으로 불가 |
| 보안/안전 이슈 | 심각한 취약점 발견 |

### 2.2 BLOCK 시 조치

```
1. 즉시 작업 중단
2. 현재 상태 저장
3. Manager에게 BLOCK 보고
4. Manager 판단 대기
5. 판단에 따라 진행
```

---

## 3. 작업 완료 후 보고

> 작업은 완료하되, Manager에게 **알려야** 하는 상황

### 3.1 보고 상황

| 상황 | 예시 |
|------|------|
| 예상보다 규모 큼 | Task가 예상보다 복잡 |
| 경미한 이슈 발견 | 작은 개선점 발견 |
| 문서 오류 발견 | 오타, 불일치 |
| 일정 지연 예상 | 예상보다 시간 필요 |

### 3.2 보고 시 조치

```
1. 작업 완료
2. 이슈 기록
3. Manager에게 보고
4. 필요시 추가 조치
```

---

## 4. 에스컬레이션 형식

### 4.1 BLOCK 보고 형식

```markdown
## Issue: [제목]

- 유형: BLOCK
- 보고자: [역할]
- 일시: YYYY-MM-DD HH:MM
- Task: TASK-XXX (해당 시)

### 상황
[구체적 상황 설명]

### 문제점
[무엇이 문제인지]

### 시도한 것
[이미 시도한 해결책]

### 필요한 판단
[Manager에게 필요한 결정]

### 제안 (있으면)
[해결책 제안]
```

### 4.2 보고 형식

```markdown
## Report: [제목]

- 유형: INFO / WARN
- 보고자: [역할]
- 일시: YYYY-MM-DD HH:MM
- Task: TASK-XXX (해당 시)

### 내용
[보고 내용]

### 영향
[예상되는 영향]

### 제안 (있으면)
[조치 제안]
```

---

## 5. 역할별 에스컬레이션 권한

| 역할 | BLOCK 가능 | 보고 대상 |
|------|:----------:|----------|
| Planner | ✅ | 요구사항 모호, 충돌 |
| Architect | ✅ | 기술적 불가, 규모 초과 |
| Developer | ✅ | 구현 불가, 기술 부족 |
| Reviewer | ✅ | 아키텍처 문제, 심각한 이슈 |
| QA | ✅ | 검증 불가, 조건 모호 |

---

## 6. 에스컬레이션 금지 사항

- ❌ 판단 가능한 것을 에스컬레이션
- ❌ 문서에 답이 있는 것을 에스컬레이션
- ❌ 개인 선호를 이유로 에스컬레이션
- ❌ BLOCK 없이 작업 중단
- ❌ 보고 없이 예외 처리

---

## 7. 에스컬레이션 응답 시간 기대치

| 유형 | 기대 응답 |
|------|----------|
| BLOCK (긴급) | 즉시 ~ 1시간 |
| BLOCK (일반) | 4시간 이내 |
| 보고 (WARN) | 24시간 이내 |
| 보고 (INFO) | 다음 체크인 |

---

## 8. 에스컬레이션 흐름도

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  이슈 발생                                                      │
│      ↓                                                          │
│  문서에 답이 있는가?                                            │
│      ├── Yes → 문서대로 처리                                    │
│      └── No ↓                                                   │
│                                                                 │
│  내 권한으로 판단 가능?                                          │
│      ├── Yes → 판단 후 진행 (필요시 보고)                        │
│      └── No ↓                                                   │
│                                                                 │
│  즉시 중단 필요?                                                │
│      ├── Yes → BLOCK 에스컬레이션                               │
│      └── No → 작업 완료 후 보고                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
