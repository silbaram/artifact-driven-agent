## 4. Game 화면 (HUD)

### 레이아웃

```
┌─────────────────────────────────────────┐
│ [체력바]                   [미니맵]     │
│ [마나바]                                │
│                                         │
│                                         │
│              [게임 영역]                │
│                                         │
│                                         │
│ [스킬1][스킬2][스킬3]    [인벤토리]     │
└─────────────────────────────────────────┘
```

### HUD 요소

| 요소 | 위치 | 표시 내용 | 이벤트 연동 |
|------|------|----------|-------------|
| 체력바 | 좌상단 | HP / MaxHP | DAMAGE, HEAL |
| 마나바 | 좌상단 | MP / MaxMP | SKILL_USE |
| 경험치바 | 상단 | EXP / RequiredEXP | EXP_GAIN |
| 미니맵 | 우상단 | 주변 지형 | MOVE |
| 스킬 슬롯 | 하단 | 스킬 아이콘, 쿨다운 | SKILL_READY |
| 인벤토리 버튼 | 우하단 | 가방 아이콘 | → Inventory |

### 게임 이벤트별 UI 반응

| 이벤트 | UI 반응 |
|--------|---------|
| DAMAGE | 체력바 감소 + 화면 빨간 플래시 |
| HEAL | 체력바 증가 + 녹색 이펙트 |
| LEVEL_UP | 레벨업 팝업 + 황금빛 이펙트 |
| ITEM_GET | 아이템 획득 알림 (우측) |

## 6. Inventory 화면

### 레이아웃

```
┌─────────────────────────────────────────┐
│ [닫기]         인벤토리                 │
├─────────────────────────────────────────┤
│                                         │
│  [아이템][아이템][아이템][아이템]       │
│  [아이템][아이템][아이템][아이템]       │
│  [아이템][아이템][아이템][아이템]       │
│                                         │
├─────────────────────────────────────────┤
│  [선택된 아이템 정보]                   │
│  [사용] [버리기]                        │
└─────────────────────────────────────────┘
```

### 요소

| 요소 | 액션 |
|------|------|
| 아이템 슬롯 | 클릭 → 선택 |
| 아이템 정보 | 선택된 아이템 표시 |
| 사용 버튼 | ITEM_USE 이벤트 |
| 버리기 버튼 | ITEM_REMOVE 이벤트 |
