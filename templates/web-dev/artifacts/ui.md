## 2. 화면별 상태 정의 (필수)

### 2.1 [화면명]: /path

#### API 연동

| API | 용도 | 상태 |
|-----|------|------|
| GET /resources | 데이터 조회 | 연동 |

#### 화면 상태

| 상태 | 조건 | 표시 |
|------|------|------|
| Loading | API 호출 중 | 로딩 스피너 |
| Empty | data.length === 0 | "데이터가 없습니다" |
| Error | API 실패 | 에러 메시지 + 재시도 |
| Success | data.length > 0 | 데이터 목록 |

#### 사용자 액션

| 액션 | 트리거 | 결과 |
|------|--------|------|
| 새로고침 | 버튼 클릭 | 목록 재조회 |
| 항목 클릭 | 목록 아이템 | 상세 화면 이동 |
